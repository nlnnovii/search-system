Deteksi Korban Hanyut

Deteksi Korban Hanyut menggunakan Kamera dengan Raspberry Pi sebaga mikrokontroler yang diolah menggunakan metode YOLOv5.

Lokasi Korban Hanyut

Lokasi korban hanyut diperoleh dari hasil deteksi korban yang berada dipermukaan air sungai, sehingga ketika korban hanyut terdeteksi oleh kamera maka GPS akan mencatat longitude dan latitude dari korban tersebut.

Klasifikasi Sungai

Klasifikasi kondisi Sungai diklasifikasi menggunakan parameter debit sungai, kemiringan tanah dan curah hujan yang diolah menggunakan metode Artificial Neural Network.

Perancangan sistem yang dikerjakan yaitu terdapat sisi sensor node menggunakan kamera yang dapat mendeteksi korban hanyut mengapung di Sungai yang telah terkompatibel dengan Raspberry Pi, sensor Flowmeter untuk mengukur debit aliran arus air, sensor Inklinometer untuk mengukur kemiringan tanah, sensor Rain Gauge untuk mengukur curah hujan harian dan GPS untuk mendeteksi lokasi korban hanyut mengapung. Sensor node berfungsi untuk mengumpulkan beberapa data. Data diperoleh dari gambar dan data sensor. Data gambar akan diolah pada Raspberry Pi menggunakan metode You only look once (YOLOv5). Dan data sensor diolah pada Raspberry Pi menggunakan metode Artificial Neural Network (ANN) untuk klasifikasi data. Data yang sudah diolah dikirimkan melalui LoRa ke gateway dan disimpan di database yang selanjutnya ditampilkan di TFT LCD dan website. TFT LCD dan website akan menampilkan informasi berupa debit aliran arus air, kemiringan tanah dan curah hujan harian sehingga dapat membantu proses pencarian korban hanyut.
